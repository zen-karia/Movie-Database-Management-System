# Movie-Database-Management-System
Integrated Python & MySQL to create a program which allowed a user to store information about movies in a binary file and as well as in MySQL in the form of a table.

Modules used: os, pickle, mysql.connector, playsound, matplotlib

User can perform various operations like adding, modifying, deleting, and searching content from the file. Graphs can be plotted for comparison between the IMDB Rating and Box Office Collection of different movies.

User can also play a hit song of a particular movie if he wishes to provided that the song is present in the folder where the Python program is saved.

User can even view the information about movies in the form of a table once he saves the same in MySQL Database.
